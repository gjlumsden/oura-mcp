# Contributing to Oura Ring MCP Server

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/oura-mcp.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Install .NET 10 SDK
5. Build: `dotnet build`
6. Test: `dotnet test`

## Development

This project follows **Test-Driven Development (TDD)**:
1. Write a failing test
2. Implement the minimum code to pass
3. Refactor

## Pull Requests

- Ensure all tests pass (`dotnet test`)
- Follow existing code style and conventions
- Add tests for new features
- Update documentation if needed
- Keep PRs focused on a single change

## Reporting Issues

Use GitHub Issues with the appropriate template (bug report or feature request).

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).
